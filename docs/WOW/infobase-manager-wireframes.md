## 9. Keyboard Shortcuts

| Действие | Shortcut | Контекст |
|----------|----------|----------|
| Фокус на панель | `Ctrl+Shift+I` | Глобально |
| Обновить статусы | `Ctrl+R` | Панель в фокусе |
| Открыть Предприятие | `Enter` | Элемент выбран |

| Загрузить конфигурацию | `Ctrl+Shift+L` | Элемент выбран |

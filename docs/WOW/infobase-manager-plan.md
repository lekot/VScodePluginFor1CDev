## Риски реализации

| Риск | Митигация |
|------|-----------|
| Webview диалога привязки (#31) — сложный UI | Начать с простого списка, итеративно улучшать |
| Режим copy (#44) — изменения во время копирования | Копируется только папка конфигурации (Configuration.xml + дерево), атомарное копирование |
| Интеграция с деревом CDT 41 (#37) — coupling | Использовать события, не прямые вызовы |

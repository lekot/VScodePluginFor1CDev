## Детализация Use-Cases

### Группа: Настройка

```
┌────────────────────────────────────────────────────────────────────────────────┐
│ UC-01: Первоначальная настройка                                                │
├────────────────────────────────────────────────────────────────────────────────┤
│ Актор: Разработчик                                                             │
│ Триггер: Первое открытие панели Infobase Manager                               │
│                                                                                │
│ Основной поток:                                                                │
│ 1. Пользователь открывает панель                                               │
│ 2. Расширение проверяет наличие ibcmd                                          │
│    2a. [extend] Автодетект ibcmd                                               │
│ 3. Если не найден — показ инструкции                                           │
│ 4. Пользователь указывает путь в Settings                                      │
│ 5. Панель активируется                                                         │
│                                                                                │
│ Постусловие: ibcmd настроен, панель готова к работе                            │
└────────────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────────────┐
│ UC-01a: Автоматическое определение ibcmd                                       │

│                                                                                │
│ Поток:                                                                         │

│    - Windows: C:\Program Files\1cv8\<version>\bin\ibcmd.exe                    │
```

## Матрица Use-Case → Команда → ibcmd

| Use-Case | Команда VS Code | Вызов ibcmd |
|----------|-----------------|-------------|
| UC-02 | `infobases.create` | `ibcmd infobase create --db-path=...` |
| UC-03 | `infobases.add` | `ibcmd infobase info --config=...` (проверка) |
| UC-04 | `infobases.importV8i` | — (только парсинг файла) |
| UC-05 | `infobase.configImport` | `ibcmd infobase config import --config=... <path>` |
| UC-06 | `infobase.configExport` | `ibcmd infobase config export --config=... --out=...` |
| UC-07 | `infobase.openEnterprise` | — (запуск 1cv8.exe) |
| UC-08 | `infobase.openDesigner` | — (запуск 1cv8.exe DESIGNER) |
| UC-09 | `infobase.configCheck` | `ibcmd infobase config check --config=...` |
| UC-10 | `infobase.remove` | — (только удаление из списка) |

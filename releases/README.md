# Релизы

Сюда кладите собранный `.vsix` для установки расширения из репозитория.

Сборка: из корня репо выполните `.\build-all.bat` (инкремент версии + сборка VSIX) или `npm run package` (только сборка без смены версии). Затем скопируйте получившийся `1c-metadata-tree-vscode-*.vsix` в эту папку и закоммитьте.
